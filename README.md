### DESCRIPTION

This Project was made with Unreal Engine 5.3.2 and PaperZD ([Link](https://www.unrealengine.com/marketplace/en-US/product/paperzd)).

## CONTROLS

A - Move Left

D - Move Right

Z - Roll

Spacebar - Jump

Left Click - Attack

Right Click - Block
